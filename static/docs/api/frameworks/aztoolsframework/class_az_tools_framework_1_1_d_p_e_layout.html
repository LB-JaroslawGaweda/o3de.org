◆ m_sharePriorColumn

AZStd::vector<AZStd::pair<QWidget*, int> > AzToolsFramework::DPELayout::m_sharePriorColumn
protected

Vector containing pairs of widgets and integers, where each pair in the vector represents a unique shared column layout. Each widget in a pair will be the first widget in the shared column, while the integer in a pair represents the number of widgets in the shared column.

◆ m_widgetAlignment

AZStd::unordered_map<QWidget*, Qt::Alignment> AzToolsFramework::DPELayout::m_widgetAlignment
protected

Map containing all widgets that have special alignment. Each widget will be aligned according to its value, where 0 = LEFT align, 1 = CENTER align, and 2 = RIGHT align.
